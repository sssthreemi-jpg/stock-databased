# PRD: 대웅제약 재무팀 손익 대시보드

## 1. 개요
매월 관리하는 손익 데이터를 CSV로 업로드하면 월별 매출 추이를 선그래프로 시각화하는 웹 대시보드.

## 2. 사용자
- 대웅제약 재무팀 담당자

## 3. CSV 파일 규격
| 열 | 내용 | 예시 |
|----|------|------|
| 1열 | 구분 | 매출 / 매출원가 / 판관비 |
| 2열 | 계정명 | 전문의약품, 일반의약품, 급여 등 |
| 3~14열 | 2025-01 ~ 2025-12 월별 금액 | 억원 단위 숫자 |

## 4. 기능 요구사항

### v1.0 (현재 범위)
- **CSV 업로드**: 드래그앤드롭 또는 파일 선택으로 CSV 업로드
- **매출 선그래프**: 구분이 "매출"인 행들의 월별 합계를 선그래프로 표시
  - X축: 월 (1월 ~ 12월)
  - Y축: 금액 (억원)
  - 각 데이터 포인트에 수치 표시

## 5. 비기능 요구사항
- **단일 파일**: index.html 하나로 동작 (외부 CDN 허용)
- **디자인**: 밝고 깔끔한 라이트 테마
- **차트 라이브러리**: Chart.js (CDN)
- **한글 인코딩**: UTF-8, EUC-KR 모두 지원
- **브라우저**: Chrome, Edge 최신 버전

## 6. 화면 구성
1. **업로드 영역**: 파일 드래그앤드롭 + 클릭 업로드
2. **차트 영역**: 매출 합계 선그래프 (업로드 후 표시)

## 7. 기술 스택
- HTML5 / CSS3 / Vanilla JS
- Chart.js (CDN)
- 서버 불필요 (클라이언트 사이드 처리)
